# CppBasic-Calculator

CppBasic-Calculator is a beginner-friendly console calculator written in C++.
It performs the four basic arithmetic operations: addition, subtraction, multiplication, and division.
The code is clear, well-commented, and easy to understand, making it ideal for learning C++ fundamentals and console input handling.

******************************************************************************************************************************************

# Features

Simple, user-friendly console interface

Supports +, -, *, / operations

Clean, readable, and well-commented code

Perfect for practicing C++ basics and conditional logic

******************************************************************************************************************************************

# How to Run

Clone or download the repository

Compile the code using a C++ compiler:

g++ CppBasic-Calculator.cpp -o CppBasic-Calculator


Run the program:

./CppBasic-Calculator


Follow the prompts to perform calculations
